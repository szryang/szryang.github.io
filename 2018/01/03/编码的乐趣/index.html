Never Settle

Posted at 2018-01-03 Linux, Windows, Programming Language, Computer Science, Joy， Coding 

​ 最近才开始看到这个词，这个词是一加手机的motto，他们翻译为“不将就”。这个词深得我心，我们的一生太过于短暂，而我们却被一下生命以外的事情所困扰。所谓纷纷攘攘，皆为利来。经济高速增长，加之移动互联网的崛起，消费主义的盛行，我们都是一个个被困在“消费城市”这座囚笼里面的无脚鸟。我们必须要没日没夜的不停的飞行，才能维持住所谓的阶层，够得着所谓的房子，小孩上得起好的培训班。

​ 人生的乐趣又在哪里呢？

​ 有人说人生是一场接力赛，父母给孩子铺路，孩子努力上名校，成为牛逼公司的员工，然后孩子的孩子就能够获得一个不错的未来。

​

​ 写下了上面那些文字，但我不是为了控诉什么。生活的还是要继续，该赚的钱还是要赚，该买的房子，车子还是要买，能够上名校还是要上名校，这是一条非常保险而且平庸的道路，也是很多人包括我在内的追求。

​ 如何保持人生的乐趣？

​ 大刘的球状闪电前面写的好，找一个世界难题研究。我觉得我的乐趣其实就在我身边，在计算机领域研究的更加深入，这里面有太多的有趣的事情值得去研究，去了解。

​ 产品经理只是我职业的一条路，但是业余爱好还是coding！

​ 我的Never Settle是永不停留，说不定未来我还会去念一个学位，知识的追求带来的快乐和社会价值远远大于其他的事情，要培养孩子热爱知识，而不是热爱消费。
